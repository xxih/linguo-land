# 快速上手指南

嗨，朋友！欢迎使用  **LinguoLand**！

这是一款能让你在浏览网页时轻松学习英语的工具。现在，LinguoLand 已正式上架 Chrome 应用商店，安装和使用变得前所未有的简单。

请按照以下三个步骤，仅需 1 分钟，即可开启你的沉浸式英语学习之旅。

## 第 1 步：从 Chrome 应用商店安装

1. **点击下方链接**，前往 LinguoLand 在 Chrome 应用商店的官方页面：

   > **[点击此处前往 Chrome 应用商店安装 LinguoLand]** > _(请将此文本替换为你的插件商店实际链接)_

2. 在打开的页面右上角，点击蓝色的  **“添加到 Chrome”** (Add to Chrome) 按钮。

3. 在弹出的确认窗口中，点击  **“添加扩展程序”**。

片刻之后，LinguoLand 就会自动安装到你的浏览器中。就是这么简单！

## 第 2 步：固定扩展程序

为了方便随时使用，我们强烈建议你将 LinguoLand 固定在浏览器工具栏上。

1. 点击浏览器右上角的“扩展程序”图标（像一个拼图块）。
2. 在弹出的列表中找到 LinguoLand，点击旁边的**图钉图标**。

现在，LinguoLand 的图标就会常驻在你的浏览器工具栏上，方便你随时访问。

## 第 3 步：注册并开始体验

1. 点击工具栏上刚刚固定的  **LinguoLand 图标**，快速注册一个账户并登录。
2. 打开任何一个你常看的英文网站（比如  [The Verge](https://www.theverge.com/)  或  [Medium](https://medium.com/)）。
3. 你会发现页面上的一些单词被**高亮**了。按住  Alt  键 (Mac 用户是  Option  键) 并**单击**这些单词，即可查看释义、发音并开始学习！

### 常见问题与反馈

- **安装后，为什么网页上没有高亮单词？**
  请尝试**刷新**当前页面。LinguoLand 需要在页面加载时注入脚本。如果问题依旧，请尝试重启浏览器。
