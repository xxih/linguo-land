# 与单词交互：查询与管理

发现一个高亮的单词？LinguoLand 提供了非常简单的方式来查询和管理它。

## 查询单词释义

这是 LinguoLand 最核心的交互。

**操作：** 按住键盘上的 <kbd>Alt</kbd> 键（在 Mac 上是 <kbd>Option</kbd> 键），然后用鼠标**单击**任何你感兴趣的英文单词（无论它是否被高亮）。

**结果：** 一个精美的“单词卡片”会立刻出现在你点击的位置，为你展示这个单词的详细信息。

## 单词卡片里有什么？

单词卡片经过精心设计，为你提供最关键的信息：

1.  **单词与音标**：清晰展示单词本身和其国际音标。
2.  **中文释义**：直接、简洁的中文定义，帮助你快速理解。
3.  **状态操作按钮**：这是管理你个人词库的关键！
    - **学习**：将该单词加入“学习中”列表。下次见到它时，会显示为蓝色下划线。
    - **掌握**：将该单词标记为“已掌握”。下次它将不再被高亮。
    - **陌生**：如果一个“学习中”或“已掌握”的单词你又忘了，可以把它重新标记为“陌生”。

:::tip 词族学习
当你将一个单词（例如 `decision`）标记为“已掌握”时，LinguoLand 非常智能，它会自动将这个单词的整个**词族**（如 `decide`, `decisive`）都视为已掌握。这极大地提升了你的学习效率！
:::

## 忽略不想学的单词

有时候，你可能会遇到一些你不想学习的专有名词或代码变量。

**操作：** 在单词卡片的**状态徽章**上（例如“陌生”标签），单击一下。

**结果：** 这个单词会被加入你的“忽略列表”，LinguoLand 将永远不再高亮它。你可以在“词汇管理”页面管理你的忽略列表。
