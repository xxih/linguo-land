# 理解单词高亮

当你浏览网页时，LinguoLand 会根据你的个人词汇库，用不同的颜色高亮页面上的单词。每种颜色都代表着你对这个单词的掌握程度。

## 高亮颜色代表什么？

| 颜色和样式                                                                                                         | 状态       | 含义                                         |
| ------------------------------------------------------------------------------------------------------------------ | ---------- | -------------------------------------------- |
| <span style={{backgroundColor: 'rgba(255, 199, 0, 0.4)', padding: '2px 6px', borderRadius: '4px'}}>黄色背景</span> | **陌生**   | 这是你从未见过，或标记为“不认识”的单词。     |
| <span style={{borderBottom: '2px dotted #3b82f6', paddingBottom: '1px'}}>蓝色虚线</span>                           | **学习中** | 这是你正在学习和巩固的单词。                 |
| (无高亮)                                                                                                           | **已掌握** | 你已经掌握的单词。它们会像普通文本一样显示。 |

:::success 好消息！
当一个单词不再被高亮时，这恰恰说明你已经进步了！它已经成为了你知识体系的一部分。
:::

## 为什么我看到的单词没有高亮？

可能有以下几种原因：

1.  **你已经掌握了它们**：这是最常见的原因！
2.  **单词过于简单**：LinguoLand 会自动过滤掉一些非常基础的常用词，以避免干扰。
3.  **高亮功能被关闭**：你可以在扩展的设置中暂时关闭高亮功能。
