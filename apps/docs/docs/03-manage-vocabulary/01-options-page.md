# 设置与词库管理

LinguoLand 提供了一个功能强大的“设置与词库管理”页面，让你完全掌控自己的学习数据和扩展行为。

**如何进入？**
在浏览器扩展程序页面找到 LinguoLand，点击“选项”；或者在扩展的弹出窗口中点击“打开词汇管理”按钮。

## 词汇管理

这是你的个人词汇中心，以表格形式展示了你学习过的所有词族。

在这里，你可以：

- **查看所有词族**：清晰地看到每个词族的状态、你遇到它的次数和最后见到的时间。
- **搜索和筛选**：快速找到特定的词族，或者只查看“学习中”的词汇。
- **批量操作**：选中多个词族，一次性将它们全部标记为“已掌握”或“陌生”。
- **展开词族**：点击每个词族旁边的小箭头，可以看到该词族包含的所有单词。

## 数据导入与导出

你的学习数据完全属于你。

- **导出为 JSON**：一键将你的整个词汇库导出为 JSON 文件，方便备份或迁移。
- **从 JSON 导入**：如果你在其他设备上有备份，或者想从别处导入词库，可以使用此功能。

## 快速添加词库

想快速扩充你的“已掌握”词汇量？我们为你准备了预设词库。

- **操作**：在“快速导入词库”区域，点击“大学四六级词汇”或“高中核心词汇”等按钮。
- **效果**：LinguoLand 会立即将这些词表中的所有单词标记为“已掌握”，这样它们就不会在网页上被高亮，让你能更专注于学习真正的生词。

## 扩展设置

在这里，你可以个性化你的 LinguoLand 体验：

- **API 配置**：为高级用户提供，通常保持默认即可。
- **网站设置**：如果你不希望 LinguoLand 在某些网站（如中文网站）上运行，可以在这里将它们加入“禁用列表”。
- **AI 解析设置**：选择 AI 功能是自动运行、手动触发还是关闭。
- **忽略列表管理**：查看所有被你“忽略”的单词，并可以随时将它们移出忽略列表，让它们重新参与高亮。
